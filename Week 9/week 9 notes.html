Week 9 Notes

  • The Window Object

  • browser object model: a collection of properties and methods: contains browser and screen info
  • BOM only makes sense in browser: other environments don't work with BOM
  • global variables can be accessed with the window keyword: like this keyword
  • window.location: contains info about url of page
  • window.history: info about browser history
  • window.screen: info about screen: height, width, etc

  • The Document Object

  • document.write: writes string of text to the page
  • document.cookie: creates a cookie
  • changing cookie values: do create cookie code, will override the previous value
  • set cookie expiry date: with expire property of cookie
  • deleting cookies: make it expire in the past

  • HTML5 APIs

  • web storage: stores data on client's computer similar to cookies
  • web storage: local storage is an example
  • geolocation: api to ge tthe location of the device
  • notifications: show messages using system's notifications
  • notificaitons: you need permission
  • multimedia: can display audio and video using the audio and video tags
