Week 7 Notes

  • Function Properties and Methods

  • Length() returns the number of parameters a funciton has
  • call() is used to set the value of this inside a function to an object that is provided
  • apply() is like call() but the parameters are passed as an array
  • memoization: result caching
  • setting the cache in the funciton allows you to store results of the funcition, then display them all at once

  • Immediately Invoked Funciton Expressions

  • Anonymous function that is called as soon as it's defined
  • You can use strict mode inside an iife
  • any variable created inside an iife is deleted after the function runs
  • iife can enclose an block of code inside its own private scope
  • Functions can rewrite themselves by setting the function to different code
  • functions can return functions

  • Ajax

  • Asynchronous JavaScript XML
  • Asynchronous: program does not have to wait for the server to return data, it can do other stuff in the meantime
  • XML: xml files used to be how data was returned. Now we use JSON
  • Fetch Api: provides a global fetch() method that only has one argument
  • Response Interface: deals with the objects that is returned when the promise is fufilled
  • redirect: used to redirect to another URL
  • JSON is used to format returning data
