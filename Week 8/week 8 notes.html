Week 8 Notes

  • transitions/animations

  • have to have a change
  • transition always goes in the base state
  • all: anything that can be transitioned
  • not everything can be animated
  • best animations in web are subtle
  • animations should not be distractions, they can be distractions very easily
  • translation: moving elements' position ex: transform: translate(45px, -45px)
  • scaling: scales the element to the specified size
  • scaling: center stays the same
  • rotation: rotates ex: transform: rotate(10deg)

  • Canvas, SVG, and Drag and Drop

  • Canvas: used to draw wahtever you want
  • must set width and height in the canvas element, not in the CSS
  • fillstyle: fills in shape
  • save Canvas: canvasName.toDataURL("image.png")
